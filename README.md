This repository contains the projects completed during my Machine Learning Internship at CodSoft.
Each task focuses on solving real-world problems using machine learning techniques.

## 📂 Tasks Overview

---

### 📨 Task 1: Spam SMS Detection

* **Status:** Completed ✅
* **Models Used:** Logistic Regression, Naive Bayes, Random Forest
* **Accuracy:** 97.6%
* **Key Metric:** High precision and recall for spam detection
* **Final Model:** Random Forest

📌 Built a text classification system using NLP techniques to accurately classify messages as spam or ham.

---

### 📉 Task 2: Customer Churn Prediction

* **Status:** Completed ✅
* **Models Used:** Logistic Regression, Random Forest
* **Accuracy:** 86.5%
* **Key Metric:** Balanced performance across churn classes
* **Final Model:** Random Forest

📌 Developed a predictive model to identify customers likely to leave based on behavioral and account data.

---

### 💳 Task 3: Credit Card Fraud Detection

* **Status:** Completed ✅
* **Models Used:** Logistic Regression, Random Forest
* **Accuracy:** 93% (Logistic Regression), 99% (Random Forest)
* **Key Metric:** Recall for fraud detection (Logistic Regression: 0.80)
* **Final Model:** Logistic Regression

📌 Built a fraud detection system on an imbalanced dataset, focusing on identifying fraudulent transactions effectively.
