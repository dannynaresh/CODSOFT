# Task 3 - Credit Card Fraud Detection

## 📌 Objective

Build a machine learning model to detect fraudulent credit card transactions.

---

## 📊 Dataset

The dataset contains transaction details such as amount, merchant, location, and fraud labels.

---

## ⚙️ Models Used

* Logistic Regression
* Random Forest

---

## 📈 Results

### Logistic Regression

* Accuracy: 93%
* Recall (Fraud): 0.80
* Successfully detects fraudulent transactions

### Random Forest

* Accuracy: 99%
* Recall (Fraud): 0.00
* Failed to detect fraud due to class imbalance

---

## 🧠 Conclusion

Although Random Forest achieved higher accuracy, it failed to identify fraud cases. Logistic Regression performed better by correctly detecting fraudulent transactions, making it the preferred model.
